Core Features

Everything your community needs to thrive.

📊

Dual XP System

Separate Text XP & Voice XP pools. Steep exponential curve for dedicated students. 1-min text cooldown, 60-min voice awards. AFK channels excluded.

🎫

Ticket System

Auto claim embeds, claim/unclaim with point tracking, post-ticket rating via DM with 1hr expiry. Bilingual rating messages.

🛡️

Moderation

Kick, ban, timeout with bilingual duration support. Audit log detection, per-channel anti-spam with silent XP freeze. Full embed logging.

💰

Economy

Coins, bank, deposit/withdraw, rob, raid with cooldown, PvP coinflip. Automatic wealth tax above threshold. Full setup command.

📚

Study Mode

Toggle quiet zone on voice channels. Denies speak for @everyone, renames channel with 📚 prefix. Full restore on disable. Owner detection compatible.

🎨

Level Cards

900×300px canvas cards with lunar background, tactical HUD overlay, dual progress bars. Milestone color swap every 10 levels. Rank titles displayed.

Bot Commands

Searchable & filterable. Type to find any command.

🤖 General

  • -prefix Set custom prefix
  • -status Configure bot presence

📊 Level System

  • -rank View your level card
  • -top voice Voice XP leaderboard
  • -top text Text XP leaderboard
  • -setxpchannel Route level-up cards
  • -setuplevelsystem Initialize level system

🎫 Tickets

  • -ticket Open a ticket
  • -close Close current ticket

🛡️ Moderation

  • -kick Kick a member
  • -ban Ban a member
  • -timeout Timeout with duration
  • -untimeout Remove timeout
  • -setlogtimechannel Set timeout log channel
  • -setantispam Toggle anti-spam per channel

💰 Economy

  • -bal View XP, coins, bank
  • -deposit Deposit coins to bank
  • -withdraw Withdraw coins
  • -rob 50/50 rob attempt
  • -raid 40% success raid
  • -coinflip PvP coin flip
  • -setup Configure tax & XP mode

📚 Study Mode

  • -studymode on Enable quiet zone
  • -studymode off Disable & restore
  • -studymode status Check current state
